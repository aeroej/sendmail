Issues Classification Demo Tool

Summary

        When user enters User Manual and Issue Data link on the Issues Claasification homepage, 
        the system crawls data of the web links, classifies issues by requirements.
        Then user gets an e-mail that is attached a zip file of Issue Classification 
        and result table web link which arranges issue data classified by requirements.

Introduction

        First, make sure all fields are entered and valid.
        Enter Project Name, User Manual and Issue Data links, and your E-mail address.
        Start Number and End number of Issue are basically 1 and 10 if you don't change.
        
        Second, click the send button.
        The system will start crawling web link data to html documents. 
        This progress will take a minute.
        

        Third, check an e-mail attached a zip file that is sended to your e-mail address.  
        The zip file has issue data links which is html documents.
        There is User Manual, Issue Data and Result links in the main of the e-mail.
        
        Finally, click the Result link on the e-mail to check arranged results.  
        There is a table that represent arranged results of crawling and classifing issue data.
        You can check Document name, the number of issue and html web link of issue data.
